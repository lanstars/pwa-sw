PWA
===========

###How does it work?###
增强型网页应用（Progressive Web Apps，PWAs），是web技术里一个令人激动的革新。PWAs集合了一系列技术，可以使web app有和原生移动app相同的功能。

###Browser compatibility###

The script should work fine on the following browsers:

* Firefox 3.5+
* Google Chrome
* Opera 12+
* IE9+
* Safari 6+
